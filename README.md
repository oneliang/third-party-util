# Get started
### Dependencies
```
https://github.com/oneliang/third-party-lib
```

### Download and Build
```
git clone https://github.com/oneliang/third-party-lib
cd third-party-lib
gradle build
gradle publishToMavenLocal
git clone https://github.com/oneliang/third-party-util
cd third-party-util
gradle build 
```

### Use for eclipse
```
gradle eclipse
```
